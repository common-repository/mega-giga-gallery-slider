== Description ==
Selectively makes your galleries to sliders, with options to customize it as u wish.
Pretty easy options on sreenshot-1. Thumbnails can be used instead of dots in your slider like on screenshot-3. After you setuped it just make gallery with number of columns that u chose and it will work like a slider on screenshot-2. After that you can style it as u wish with css. :)
